# 🧪 Mollab - Plataforma Educacional de Química

## 📚 Sobre o Projeto

**Mollab** é uma plataforma educacional interativa desenvolvida para facilitar o aprendizado de química através de conteúdo dinâmico e ferramentas educacionais. O projeto combina videoaulas, explicações textuais e quizzes interativos para proporcionar uma experiência de aprendizado moderna e envolvente.

**Tecnologias utilizadas:**
- React - Framework JavaScript para construção da interface
- Vite - Ferramenta de compilação rápida e otimizada
- Tailwind CSS - Estilização responsiva e moderna
- Firebase - Servidor de dados e autenticação (quando configurado)

---

## 🚀 Começando

### Pré-requisitos

Antes de começar, você precisa ter **Node.js** instalado em sua máquina:
- Baixe em: [https://nodejs.org/](https://nodejs.org/) (versão LTS recomendada)

### Instalação e Execução

1. **Acesse a pasta do projeto:**
   ```bash
   cd c:\Users\dudan\OneDrive\Documentos\mollab
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Abra no navegador:**
   - Acesse o link exibido no terminal (geralmente `http://localhost:5173`)
   - A página será recarregada automaticamente ao fazer alterações no código
